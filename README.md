# Jamaica Import Calculator - Frontend

Professional import cost calculator for Jamaica Customs.

## Features

- Real-time import cost calculation
- Support for all product categories
- Vehicle engine-based SCT calculation
- De minimis exemption detection
- Mobile responsive design

## Tech Stack

- React (via CDN)
- Vanilla JavaScript
- CSS3 with CSS Variables
- FastAPI Backend

## Setup

1. Update API_URL in index.html to point to your backend
2. Deploy to Netlify or any static host
